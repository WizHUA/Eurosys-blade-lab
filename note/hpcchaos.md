- cpu: CPU资源负载故障
  - fullload: 制造CPU负载
    - `--cpu-percent`: CPU占用率 (0-100)
    - `--cpu-count`: 影响的CPU核心数量
    - `--cpu-list`: 指定CPU核心 (例如 0,3 或 1-3)
    - `--timeout`: 故障持续时间 (秒)
- mem: 内存资源负载故障
  - load: 制造内存负载
    - `--mode`: 模式 (ram:物理内存, cache:页缓存)
    - `--mem-percent`: 内存占用百分比 (0-100)
    - `--reserve`: 保留内存大小 (MB)
    - `--timeout`: 故障持续时间 (秒)
- disk: 磁盘I/O和空间相关故障
  - burn: 增加磁盘读写IO负载
    - `--read`: 启用读负载
    - `--write`: 启用写负载
    - `--path`: 目标路径 (默认为 /)
    - `--size`: 读写块大小 (MB)
    - `--timeout`: 故障持续时间 (秒)
  - fill: 填充指定目录路径
    - `--path`: 目标路径 (默认为 /)
    - `--size`: 填充大小 (MB)
    - `--percent`: 按百分比填充磁盘
    - `--timeout`: 故障持续时间 (秒)
- network: 网络相关故障（延迟、丢包等）
  - corrupt: 损坏网络包
    - `--percent`: 报文损坏率 (%)
    - `--interface`: 网卡设备 (例如 ens33)
    - `--destination-ip`: 目标IP
  - delay: 网络延迟
    - `--time`: 延迟时间 (ms)
    - `--offset`: 延迟时间波动范围 (ms)
    - `--interface`: 网卡设备
  - dns: 篡改DNS解析
    - `--domain`: 目标域名
    - `--ip`: 解析到的目标IP
  - dns_down: 使DNS不可用
    - `--allow_domain`: 允许正常解析的域名
  - drop: 丢弃网络包 (防火墙)
    - `--source-ip`: 源IP
    - `--destination-ip`: 目标IP
    - `--destination-port`: 目标端口
  - duplicate: 复制网络包
    - `--percent`: 报文重复率 (%)
    - `--interface`: 网卡设备
  - loss: 网络丢包
    - `--percent`: 丢包率 (%)
    - `--interface`: 网卡设备
  - occupy: 占用端口
    - `--port`: 需要占用的端口
    - `--force`: 强制杀死已占用该端口的进程
  - reorder: 网络包乱序
    - `--percent`: 乱序的报文比例 (%)
    - `--gap`: 报文间的间隔
    - `--time`: 延迟时间 (ms)
- process: 进程相关故障
  - kill: 杀死进程
    - `--process`: 进程名关键字
    - `--process-cmd`: 进程启动命令关键字
    - `--pid`: 进程ID
    - `--signal`: 信号量 (例如 9, 15)
  - stop: 暂停进程 (假死)
    - `--process`: 进程名关键字
    - `--pid`: 进程ID
- file: 文件系统操作故障
  - add: 添加文件或目录
    - `--filepath`: 文件或目录的完整路径
    - `--content`: 文件内容
    - `--directory`: 声明路径是目录
  - append: 追加文件内容
    - `--filepath`: 目标文件路径
    - `--content`: 要追加的内容
  - chmod: 修改文件权限
    - `--filepath`: 目标文件路径
    - `--mark`: 权限标记 (例如 777)
  - delete: 删除文件
    - `--filepath`: 目标文件路径
    - `--force`: 强制删除, 不可恢复
  - move: 移动文件
    - `--filepath`: 源文件路径
    - `--target`: 目标目录
- script: 脚本执行相关故障
  - delay: 脚本函数执行延迟
    - `--file`: 目标脚本文件路径
    - `--function-name`: 目标函数名
    - `--time`: 延迟时间 (ms)
  - exit: 脚本函数直接退出
    - `--file`: 目标脚本文件路径
    - `--function-name`: 目标函数名
    - `--exit-code`: 退出码
    - `--exit-message`: 退出信息
- time: 系统时间相关故障
  - travel: 时间旅行
    - `--offset`: 时间偏移量 (例如 5m30s, -2h)
    - `--disableNtp`: 是否禁用NTP同步
- systemd: Systemd服务管理相关故障
  - stop: 停止systemd服务
    - `--service`: 服务名称
